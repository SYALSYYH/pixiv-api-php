pixiv-api-php
======
_Pixiv API for PHP (with Non-Auth supported)_

## /v1/ranking/all

```php
$api = new PixivAPI();
echo $api->image_sizes('px_128x128,px_480mw,large')
    ->mode('daily_r18') //Also 'daily'
    ->per_page(10)
    ->page(1)
    ->profile_image_sizes('px_170x170,px_50x50')
    ->ranking();
```

## /v1/search/works.json

```php
$api = new PixivAPI();
echo $api->image_sizes('px_128x128,px_480mw,large')
    ->mode('exact_tag')
    ->per_page(30)
    ->page(1)
    ->profile_image_sizes('px_170x170,px_50x50')
    ->q('ラブライブ')
    ->search();
```
